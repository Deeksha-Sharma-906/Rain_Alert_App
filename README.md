# Rain_Alert_App
A Python-based GUI application that fetches real-time weather forecasts using an external API and intelligently alerts users about the possibility of rain in the next 12 hours based on weather condition analysis.

Open Weather App: https://api.openweathermap.org/data/2.5/forecast
API Key: https://home.openweathermap.org/api_keys (Sign in to get the key)
JSON Viewer: https://jsonviewer.stack.hu/
Online Weather Details: https://www.ventusky.com/
